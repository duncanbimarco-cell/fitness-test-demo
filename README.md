# 八年级体测成绩换算工具

一个面向北京八年级（初二）学生的体质健康测试成绩换算工具。输入各项体测成绩，自动算出单项得分、学年总分、综合等级，并以雷达图直观展示能力分布，同时折算为北京中考过程性考核体测分。

## 功能

- 七项体测成绩录入（身高体重 BMI、肺活量、50米跑、坐位体前屈、立定跳远、引体向上/仰卧起坐、1000米/800米跑）
- 实时计算单项得分与等级（优秀/良好/及格/不及格）
- 学年总分计算（满分 100，含引体向上/耐力跑加分，可达 120）
- 七维能力雷达图可视化
- 折算北京中考过程性考核体测分（满分 10 分）

## 数据来源

评分标准依据教育部《国家学生体质健康标准（2014 年修订）》八年级官方评分表。

## 使用方式

直接用浏览器打开 `index.html` 即可使用，无需安装。手机、平板、电脑均可。

## 说明

本工具为教学辅助用途，计算结果仅供参考，正式成绩以学校实测为准。三类六档赋分的及格档具体分值，各区实施细则可能略有差异。
